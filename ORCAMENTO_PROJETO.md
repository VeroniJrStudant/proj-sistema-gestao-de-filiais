# Proposta de Orçamento para o Projeto de Sistema de Gestão de Filiais

## Escopo do Projeto
Este projeto visa desenvolver um sistema de gestão eficiente para as filiais da empresa, facilitando a administração e o controle das operações. O sistema incluirá funcionalidades como gestão de inventário, controle financeiro e relatórios de desempenho.

### Objetivos Específicos
- Implementar um sistema que integre dados de múltiplas filiais.
- Garantir a segurança das informações e conformidade com normas de proteção de dados.

## Cronograma
O projeto será dividido nas seguintes fases:
1. **Planejamento**: 2026-05-01 a 2026-05-15
2. **Desenvolvimento**: 2026-05-16 a 2026-08-30
3. **Testes**: 2026-09-01 a 2026-09-15
4. **Implementação**: 2026-09-16 a 2026-09-30

## Entregáveis
- Documento de requisitos
- Protótipo do sistema
- Sistema finalizado
- Relatórios de testes

## Termos de Pagamento
Os pagamentos serão feitos em três parcelas: 
1. 30% na fase de planejamento
2. 40% na fase de desenvolvimento
3. 30% na entrega do sistema finalizado
